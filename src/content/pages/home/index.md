## Desenvolvimento de Software para Linux Embarcado

Para dispositivos que demandam interfaces ricas, alta conectividade, gateways e necessidade de integração com periféricos.

- Personalização e otimização de imagens
- Desenvolvimento de interfaces gráficas para **displays**
- Otimização do **tempo de Boot.**
- Desenvolvimento de **BSPs e drivers**
- Modificação e configuração de **Device-Tree**
- Preparação do **ambiente de programação e compilação cruzada**
- **Inicialização de aplicações via Systemd e SysVinit**

## Desenvolvimento de Firmware para Microcontroladores

Para dispositivos que demandam respostas em tempo real e baixo consumo

- Suporte para Microcontroladores STM32, Nordic, NXP, ESP32
- Desenvolvimento de soluções sem Baremetal ou RTOS.
- Redução de footprint em memória e otimização do consumo de energia.
- Desenvolvimento de drivers e bibliotecas de dispositivos.
- Configuração de periféricos como UART, I2C, PWM, GPIO, DMA, ADC, USB e CAN.
- Desenvolvimento com camadas de abstração - HAL
- Analisadores estáticos de código e testes unitários.
- Versionamento de código e deploy seguro.
