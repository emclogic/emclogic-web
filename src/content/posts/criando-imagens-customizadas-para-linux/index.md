No mundo do Linux Embarcado, a criação de imagens customizadas é um desafio para nós desenvolvedores e é aí que o Yocto Project se destaca, ganhando cada vez mais força no mercado por sua abordagem prática e automatizada na hora do desenvolvimento.  
Com uma estrutura intuitiva de **metadados** e **camadas**, o Yocto facilita o processo de criação de imagens que contenham configurações específicas para nossos projetos.  
Seu sistema de build, equipado com **receitas prontas** e suporte da comunidade, _abrange uma ampla gama de arquiteturas e plataformas_.  
E o que realmente facilita o trabalho do desenvolvedor é a reutilização eficiente de receitas, permitindo uma personalização descomplicada através da **herança de classes e anexação de novos comandos** (via .bbappend).  
Com o Yocto Project, a criação de imagens Linux customizadas especialmente para o projeto em questão se torna uma tarefa mais simples e acessível para todos os desenvolvedores envolvidos.

Por isso nesse Webinar com o Portal Embarcados discutimos como o desenvolvimento Linux com Yocto Project pode ser utilizado em seus projetos para esse processo de automatização.
