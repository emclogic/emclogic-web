Por que aprender Linux deveria ser sua prioridade como profissional da área de tecnologia?  
Sendo um Sistema Operacional de código aberto que está no mercado há mais de 30 anos, o Linux só tem ganhado mais espaço na área de Sistemas Embarcados.  
Assim, ele se destaca dos outros Sistemas Operacionais mais famosos por conta de sua licença de código aberto ([GNU GPL](https://www.emc-logic.com/entendendo-a-licenca-gpl-e-seu-impacto-no-linux-embarcado-e-na-comunidade-open-source/)), que possibilita projetos customizados, uma vantagem enorme para quem trabalha com Sistemas Embarcados.  
  
Além disso, o Linux conta também com uma grande comunidade global que contribui constantemente em larga escala através de plataformas como o GitHub, tornando o sistema mais seguro e eficiente. Desenvolvedores e entusiastas podem contribuir para o código fonte com alterações, relatórios e correções de bugs, entre outros, tornando o processo de melhoria muito mais rápido e eficiente.  
Esse sistema colaborativo nos traz, também, várias distros (ou distribuições) do Sistema Operacional, que podem ser manipuladas e utilizadas conforme as diretrizes que o projeto exige. O Ubuntu talvez seja a distro mais conhecida por conta de sua fácil utilização e interface intuitiva, mas outras distribuições como Fedora, Debian, Linux Mint, entre outras, possuem suas características específicas e adaptáveis.

Todas essas vantagens fazem com que o Linux esteja crescendo cada vez mais no mercado nacional e Internacional, criando portanto a necessidade de profissionais qualificados no mercado.

Juntamente com o Portal Embarcados, discutimos nesse Webinar a funcionalidade dos sistemas Linux, assim como a sua importância no mercado para todos que possuem interesse na área.  
Nessa conversa, focamos nos tópicos:  
\- Linux Desktop/Server vs Linux Embarcado  
\- Perfil profissional para trabalhar com Linux Embarcado  
\- Exemplos de aplicações com Linux Embarcado  
\- Oportunidade para desenvolvedores Linux Embarcado no Brasil
